# Medidi (MediLink) - Smart Medication Dispenser App

## Quick Start

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the Development Server

```bash
npx expo start --tunnel
```

This will start the Expo development server and display a QR code in your terminal.

### 3. Install Expo Go App

- **iOS**: Download [Expo Go](https://apps.apple.com/app/expo-go/id982107779) from the App Store
- **Android**: Download [Expo Go](https://play.google.com/store/apps/details?id=host.exp.exponent) from Google Play

### 4. Scan QR Code

1. Open the **Camera** app on your phone
2. Point your phone at the QR code displayed in your terminal
3. The app will load and run on your device

That's it! You're ready to test the app on your phone.
